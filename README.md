# NEXTY PRO - منصة التعليم الذكي المتقدمة

نسخة متطورة واحترافية من منصة NEXTY لتعليم مهارات ICDL، تتميز بتصميم عصري (Modern UI/UX) ورسوم متحركة تفاعلية.

## المميزات المتقدمة:
- **تصميم Modern UI:** استخدام الـ Glassmorphism والـ Radial Gradients لتجربة بصرية مذهلة.
- **رسوم متحركة (Animations):** تأثيرات دخول سلسة (Fade-in) وحركات تفاعلية للعناصر عند التمرير.
- **تجربة مستخدم (UX) محسنة:** نظام تنقل ذكي، بطاقات دروس تفاعلية، واختبارات بأسلوب عصري.
- **دقة المحتوى:** الالتزام الكامل بالمعايير الأكاديمية (أهداف معرفية، مهارية، ووجدانية).
- **تقنيات نظيفة:** مبني بالكامل باستخدام HTML5, CSS3, و Vanilla JavaScript بدون مكتبات ثقيلة.

## هيكلية المشروع:
- `index.html`: الواجهة الرئيسية العصرية.
- `/pages`: صفحات الدروس والاختبارات المتقدمة.
- `/css/style.css`: نظام التنسيق المتقدم والرسوم المتحركة.
- `/js/lessons-data.js`: قاعدة بيانات المحتوى التعليمي المطور.

## التشغيل:
افتح `index.html` في المتصفح أو ارفعه مباشرة إلى GitHub Pages.
