# Uoni - منصة تعليم الذكاء الاصطناعي المتطورة

![Uoni Banner](https://img.shields.io/badge/Uoni-AI_Education-7dd3fc?style=for-the-badge&logo=openai&logoColor=white)

**Uoni** هي منصة تعليمية حديثة مصممة خصيصاً لتمكين الأفراد من إتقان مهارات الذكاء الاصطناعي (AI) بأسلوب تفاعلي وعصري. تم تحويل هذا المشروع من Nexty Advanced ليكون متخصصاً في تقنيات المستقبل مع هوية بصرية فريدة تعتمد على **الوضع الداكن (Dark Mode)** واللون **الأزرق الفاتح (Baby Blue)**.

## 🌟 المميزات الرئيسية

- **تصميم عصري وجذاب:** واجهة مستخدم متطورة تدعم الوضع الداكن لراحة العين أثناء التعلم الطويل.
- **مسارات تعليمية متخصصة:** تغطي مفاهيم الذكاء الاصطناعي، تعلم الآلة، والذكاء الاصطناعي التوليدي.
- **اختبارات ذكية:** نظام تقييم فوري وتفاعلي لقياس مستوى الاستيعاب والتقدم.
- **تحديد المستوى:** اختبار ذكي يحلل خلفية المستخدم ويوجه المسار التعليمي الأمثل له.
- **تجربة مستخدم سلسة:** تخطيط محسن يركز على سهولة الوصول للمعلومة والتركيز على المحتوى.

## 🛠 التقنيات المستخدمة

- **HTML5 & CSS3:** مع استخدام متغيرات CSS لتسهيل إدارة السمات اللونية.
- **JavaScript (Vanilla):** لإدارة التفاعلات، الاختبارات، وعرض البيانات الديناميكية.
- **Google Fonts:** استخدام خطوط "Noto Kufi Arabic" و "Inter" لتجربة قراءة مثالية.

## 🚀 كيف تبدأ؟

1. قم بفتح ملف `index.html` في متصفحك.
2. ابدأ باختبار **تحديد المستوى** لمعرفة من أين تبدأ.
3. تصفح **المسارات التعليمية** واختر ما يناسب شغفك.
4. خض **التحديات الذكية** لاختبار مهاراتك الجديدة.

## 📁 هيكل المشروع

```text

```
